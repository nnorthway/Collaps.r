Flat UI Collapsing & Sliding Nav

Displayed above is a collapsing and sliding navigation menu. Built using CSS & JavaScript, what happens here is this:
When the button is clicked, it runs the JavaScript function. The function selects the button and changes it's 'onClick' function from either 'menuOpen()' or 'menuClose()' to the opposite. It then selects the menu and changes it's class from either 'open' or 'closed' to the opposite.

The mechanics:

So, the CSS is really what makes the menu slide and work. The .open class is it's default state. The id holds all of the default stylings and the .open class controls it's display and slide in. The '.closed' class slides the menu out and hides it out of view. The JavaScript controls the state of the menu's class, and the button controls when the JavaScript function is called.

(relevant) CSS:

So, the .open class is the default state. No margin set because the effect is dependent on having a negative margin. The transition makes it slide rather than just appear. The .closed class hides the nav just offscreen by using a percentage instead of a px or em amount. The transition makes it slide.

.open {
margin-left: 0;
transition: all 1s;
-webkit-transition: all ease-out 1s;
-moz-transition: all ease-out 1s;
-o-transition: all ease-out 1s;
}

.closed {
margin-left: -100%;
transition: all 1s;
-webkit-transition: all ease-in 1s;
-moz-transition: all ease-in 1s;
-o-transition: all ease-in 1s;
}

JavaScript

Essentially, as stated above, on menuClose() it selects the button and changes it's function to menuOpen(). It then selects the nav menu and changes it's class to be in a hidden state. On menuOpen(), it does the exact opposite. Nothing really fancy happening here.

function menuClose() {//collapses the navigation menu

document.getElementById("toggle").setAttribute
("onClick","menuOpen()");
//changes the button's function from menuClose() to menuOpen()

document.getElementById("menu").setAttribute
("class","closed");

//changes the class of the nav and therefore hides the menu! Violia!

}



function menuOpen() {//shows the navigation menu

document.getElementById("toggle").setAttribute
("onClick","menuClose()");

//Changes the button's function from openMenu() to closeMenu()

document.getElementById("menu").setAttribute
("class","open");

//changes the class of the nav and therefore shows the menu! Violia!

}
